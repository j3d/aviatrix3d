File loader for the Virtual Terrain Project (VTP) file formats.

File loaders are built over the top of the basic parsing framework defined in the {@link org.j3d.loaders.vterrain} package to handle the parsing.

Implementation Notes

The VTP project homepage is http://www.vterrain.org.

VTP models do not contain anything other than terrain data.

Required Support JARS and packages

  • {@link org.j3d.loaders.vterrain}, j3d-org-loaders-vterrain.jar
  • {@link org.j3d.renderer.aviatrix3d.loader}, aviatix3d-all.jar
